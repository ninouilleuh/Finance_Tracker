# 💰 Personal Finance Tracker

A command-line personal finance tracker written in Python.

This project is part of my journey to becoming a backend developer and eventually moving to Japan.

## Features

Current features:

- Add expenses
- Save expenses to a file
- Read previous expenses
- Input validation
- Error handling

## Planned Features

- Add income
- Expense categories
- Savings goals
- Debt tracking
- Search expenses
- Monthly reports
- Statistics
- Settings
- JSON data storage
- Currency conversion (EUR ⇄ JPY)

## Technologies

- Python

## How to Run

```bash
python main.py
```
